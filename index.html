HttpEventstore

HttpEventstore

HttpEventstore is a HTTP connector to the Greg's Event Store. The master branch is curently targeting EventStore 3.x, for EventStore 2.x version check EventStore2.x branch.

Supported version's of Event Store

To take advantage of all the functionality offered by our gem the minimum recommended version of Event Store is 3.0
